# 00. 팀 베이스라인 대비 변경 기록 (공유용)

> 이 문서는 **팀 결과물(본체)을 디벨롭하면서 무엇을, 왜 바꿨는지**를 팀원과 투명하게
> 공유하기 위한 기록입니다. 분석을 새로 시작한 게 아니라, 팀 분석을 계승·보정·확장했습니다.
> 숫자가 발표 자료와 달라지는 부분이 있으므로, 합의 후 반영 여부를 결정하면 됩니다.

## 한눈에 보기

| 구분 | 팀 베이스라인 | 디벨롭 레이어 | 영향 |
| --- | --- | --- | --- |
| serious 정의 | 코드 플래그=사망/입원/생명위협 3개 / 도넛=+병원방문 (불일치) | FDA 공식 8개 카테고리로 통일 + severe 상위티어 분리 | 심각율 숫자 변동 |
| 식품군 | Dietary Conv를 건기식에 병합 | 별도 분리 유지 | 숨은 1위 신호 노출 |
| 나이 환산 | `int()` 절삭 → 12개월 미만 영아 0세로 뭉개짐 | 분수 보존(3개월=0.25) | 영아 정밀도 ↑ |
| 분석 단위 | 식품군·브랜드 시각화(수작업 이상치 발견) | 브랜드 자동 신호탐지(갈래 B) | 재현·확장 |
| 산출물 | 노트북 + PPT | 코드+docs+figures+대시보드 | 인터랙티브화 |

## 보정 1 — serious 정의 통일 (가장 중요)

팀 산출물에는 serious 정의가 **두 개**가 있었고 서로 달랐습니다.

- 코드의 `serious` 플래그: `DEATH / HOSPITALIZATION / LIFE THREATENING` 3개만 →
  실제 데이터에서 **가장 많은 결과 카테고리인 `OTHER SERIOUS (IMPORTANT MEDICAL EVENTS)`
  (36,837건)와 `SERIOUS INJURIES`(3,606건)를 누락**했습니다. (과소집계)
- 도넛 차트의 "43.9% 심각": 위에 더해 **`병원방문(VISIT)`을 포함**했습니다. (과대집계)

FDA CAERS 결과 분류상 '병원/ER 방문'은 *medically attended*일 뿐 *serious*가 아닙니다.
그래서 **FDA 공식 serious 8개 카테고리**(DEATH, LIFE THREATENING, HOSPITALIZATION,
DISABILITY, CONGENITAL ANOMALY, REQ. INTERVENTION, OTHER SERIOUS, SERIOUS INJURIES)로
단일화했습니다.

| 정의 | 심각율 |
| --- | --- |
| 팀 코드 플래그 (death/hosp/life) | 약 26.5% |
| 팀 도넛 (+ 병원방문) | 43.9% |
| **FDA serious (이 작업)** | **64.2%** |
| **severe 상위티어 (이 작업)** | **26.6%** |

`OTHER SERIOUS`가 워낙 커서 FDA serious는 64.2%로 높게 나옵니다. 이 카테고리는 다소
포괄적이라, **실제 고위험 의사결정에는 `severe` 상위티어(사망/생명위협/입원/장애/선천이상,
26.6%)를 기본으로 쓰고**, FDA serious는 광의 지표로 병행합니다. 대시보드에서 두 티어를
토글할 수 있게 했습니다.

> 팀 발표 숫자(43.9%)를 유지할지, 이 통일 기준으로 갱신할지는 팀 합의 사항입니다.

## 보정 2 — Dietary Conv 분리

1차 분석에서 `Dietary Conv Food/Meal Replacements`를 건기식(Vit/Min)에 병합했더니,
이 그룹의 높은 심각율이 평균에 희석됐습니다. 분리하면 **식사대용식이 식품군 중 심각율
최상위권**으로 드러납니다(병합 시 안 보이던 신호). `figures/fig3_foodgroup_severe.png` 참고.

## 보정 3 — 나이 환산 절삭

팀 `convert_age`는 `int(age/12)` 등으로 잘라 12개월 미만이 전부 0세가 됐습니다.
분수로 보존(3 Month → 0.25)했습니다. 연령대 구간 분석에는 영향이 작지만, 영아 정밀
분석 시 의미가 있습니다.

## 유지한 팀의 결정 (계승)

- `REDACTED`(익명 처리) 항목을 결측으로 보고 브랜드 분석에서 제외 — 팀 트러블슈팅 그대로.
- 브랜드 분석 최소 신고량 ≥ 20건 — 팀 기준 계승.
- 친화적 컬럼명(brand/symptoms/...) — 팀 노트북과 동일하게.
