# Project Summary

## 프로젝트명
FDA CAERS 식품 안전 신호 탐지 프로젝트

## 핵심 문제
식품·건강기능식품 부작용 신고 데이터에서 어떤 브랜드와 식품군을 우선 조사해야 하는지 빠르게 판단하기 어렵다.

## 접근 방식
- FDA CAERS 신고 데이터를 전처리하고 serious/severe 심각도 기준을 재정의
- 식품군·연령대·증상·연도별 EDA 수행
- 브랜드별 연도 신고 건수에서 급증 신호 탐지
- 고위험 결과 비율과 급증 강도를 결합해 `watch_score` 산출
- 실제 FDA 리콜 사례와 탐지 연도를 대조해 검증

## 주요 산출물
- `README.md`: 프로젝트 설명
- `app/dashboard.html`: 설치 없이 확인 가능한 정적 대시보드
- `app/dashboard.py`: Streamlit 대시보드
- `docs/`: 문제정의, 변경기록, 신호탐지 방법, 업로드 가이드
- `src/`: 전처리, 신호탐지, 시각화 생성 코드

## 해석 주의
이 결과는 위험 확정이 아니라 조사 우선순위입니다. CAERS는 자발신고 데이터이며 판매량 분모가 없습니다.
